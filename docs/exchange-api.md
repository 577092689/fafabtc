[火币API](https://github.com/huobiapi/API_Docs/wiki)
===========
## base url
https://api.huobipro.com/

## 行情
+ GET /market/detail/merged 获取聚合行情(Ticker)
+ GET /market/detail 获取 Market Detail 24小时成交量数据

## 交易
+ GET /v1/common/symbols 查询系统支持的所有交易对及精度
